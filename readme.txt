RULE        → ACTION PROTOCOL FROM SOURCE TO DEST PORT PORTNUM
ACTION      → ALLOW | DENY
PROTOCOL    → TCP | UDP | ICMP | ANY
SOURCE      → IP | ANY
DEST        → IP | ANY
PORTNUM     → number


ALLOW TCP FROM 192.168.1.10 TO 10.0.0.5 PORT 80
DENY ICMP FROM ANY
ALLOW UDP FROM 192.168.1.0/24 TO ANY PORT 53

Valid Examples
ALLOW TCP FROM 192.168.1.10 TO ANY PORT 80
DENY UDP FROM ANY TO 10.0.0.5 PORT 53
ALLOW ICMP FROM ANY TO ANY

Invalid Examples
ALLOW TCP 192.168.1.10 TO ANY PORT 80
ALLOW ICMP FROM ANY TO ANY PORT 80
ALLOW TCP FROM ANY TO ANY PORT 70000
